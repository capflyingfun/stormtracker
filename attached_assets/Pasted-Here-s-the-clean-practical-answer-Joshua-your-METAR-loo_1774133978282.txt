🌍 Why U.S. METAR works but international fails

In the U.S.:

• Airports are dense
• ICAO codes are predictable (K + 3 letters)
• NOAA/AWC provides free METAR for all of them
• Your app can easily find the nearest one


Outside the U.S.:

• Airports may not report METAR
• Some only report TAF
• Some report METAR only during operating hours
• Some countries restrict METAR distribution
• ICAO codes don’t follow a simple pattern
• Your “nearest airport” lookup may be failing


So the issue is not the METAR API — it’s finding the correct airport.

✔ Step 2 — Find the nearest ICAO airport to the user

Use haversine distance:

function nearestAirport(lat, lon, airports) {
  let best = null;
  let bestDist = Infinity;

  for (const a of airports) {
    const d = haversine(lat, lon, a.lat, a.lon);
    if (d < bestDist) {
      bestDist = d;
      best = a;
    }
  }
  return best;
}
