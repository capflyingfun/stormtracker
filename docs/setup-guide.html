Set up AI-Written Alerts

Make your storm notifications read like a real person wrote them — using your own OpenAI key.

Difficulty: Easy ⏱️ About 10 minutes 💲 Typically under $1/month
This is completely optional. StormTracker works fully without it — you'll just get the standard alert wording. Turn this on only if you'd like your notifications rephrased into one short, natural sentence. Because it uses your own OpenAI key, the cost is billed to your account, not shared with anyone else.

  1. Create a free OpenAI account

    Go to platform.openai.com/signup and sign up (you can use Google or Apple to make it quick). If you already have one, just sign in.

  2. Add a payment method

    Open Billing and add a card, then add a small amount of credit (even $5 lasts a long time for short alert messages).

    OpenAI needs billing set up before a key will work — this is the most common reason a key gets "rejected."

  3. Create an API key

    Go to API keys → Create new secret key. Give it any name (e.g. "StormTracker") and click create.

    Your key starts with sk-... — tap Copy. Copy it now; OpenAI only shows it once.

  4. Paste it into StormTracker

    Back in the app, open ⚙️ Settings → Background alerts. Turn AI-written alerts ON, then paste your key into the Your OpenAI key box.

    Tap Test key — you want to see ✓ Key works. Then you're done; it saves automatically.

  5. That's it — enjoy natural alerts

    Your next background storm alert will be written in plain, natural language in your chosen tone (Professional, Friendly, or Humorous — set under the AI Assistant settings). The most urgent threat always comes first, and if the AI is ever unavailable you simply get the normal wording.

What does it cost?

Each alert is a tiny request (a few short lines), so it costs roughly a fraction of a cent. It only runs when you'd already be getting a notification and you've opted in — so for most people it's well under $1 per month. You can always check usage on your OpenAI billing page.

Is my key private?

Your key is stored on your device and on StormTracker's alert server (encrypted), used only to write your own notifications while the app is closed. It's never shared with other users and never shown back to anyone. To remove it, clear the key box (or turn AI-written alerts off) and it's deleted on the next sync.
